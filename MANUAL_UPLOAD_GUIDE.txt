OPTION 2: Use GitHub Desktop (Troubleshooting)
-----------------------------------------------

If GitHub Desktop isn't showing changes:

1. Close GitHub Desktop completely
2. Reopen GitHub Desktop
3. Click "Fetch origin" button
4. Check "Changes" tab again
5. If still no changes, click "Repository" > "Repository Settings" 
6. Verify the path is: Y:\AR test AI
